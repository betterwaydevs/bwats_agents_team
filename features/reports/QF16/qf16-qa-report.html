Function Detection (functions/agents/8049_call_prospect_router.xs, line 140)

// Check if agent reported all scoring failed (TASK_FAILED marker)

This runs before the transient error check (line 147+), so TASK_FAILED detection takes priority. When $agent_error is set and doesn't match transient patterns, the task status becomes "failed" (line 166).

PASS — TASK_FAILED marker in agent prompt + detection logic in function + correct status flow to "failed".
