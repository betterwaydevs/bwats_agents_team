Implementation Notes

  • The useTouchpoints hook fetches ALL touchpoints for each person (no projectId filter in the API call), then splits them client-side into current-project vs other-project sets.
  • getTouchpointsForPerson() filters by project_id === currentProjectId for the main touchpoints display.
  • getOtherProjectTouchpointsForPerson() filters by project_id !== currentProjectId for the Globe indicator.
  • The Globe icon uses opacity-50 and text-muted-foreground to visually distinguish it from active touchpoints.
  • Tooltip groups by project_name (resolved from the projects query cache via projectNameMap), falls back to "Another project" if name not found.
  • The dash (—) displays when: no current-project touchpoints, no VideoAsk response, no applicant matches, AND no other-project touchpoints.
