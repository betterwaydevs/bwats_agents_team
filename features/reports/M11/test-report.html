  • Implementation Variance: Spec (M11-dashboard-ai-assistant.md) called for using Anthropic API with @anthropic-ai/sdk npm package. Actual implementation uses Claude CLI subprocess. This is functionally equivalent and arguably better (leverages Claude Code's built-in agent system).
  • Agent Integration: Chat uses product-owner agent from the team roster, providing full access to project context via Read/Glob/Grep tools.
